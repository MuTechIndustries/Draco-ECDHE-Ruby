# Ecdhe

Allows two parties to secure a private key for symetric encryption via an unsecured line.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ecdhe'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ecdhe
